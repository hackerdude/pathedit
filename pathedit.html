Project: PathEdit
Author: David Martinez
Company:
Description:
PathEdit is an editor for PATH-style environment variables. It is useful for visually editing a path in a text file, adding, removing and rearranging items. It is designed to leave the rest of the text stream unmolested, allowing you to change the paths without worrying about the rest of the stream.
